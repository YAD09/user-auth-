Implemented Security Features:

  • Input Validation & Sanitization: Prevents malicious payload injection (e.g., XSS) via strict server and client validation.
  • SQL Injection Protection: SQLite parameterised queries strictly separate query logic from data.
  • Secure Session Management: HttpOnly, strict SameSite cookies mapping to securely managed server sessions (SQLite Store).
  • Password Security: Passwords are cryptographically hashed using generic-purpose secure hashing algorithm (bcrypt).
  • Protection Headers: Strict Content Security Policy (CSP) and multiple attack vectors managed via Helmet.js.
  • Rate Limiting: Endpoint protection against brute-force attacks on application layer.
